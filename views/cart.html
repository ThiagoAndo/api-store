Cart Routes

Cart route features and usage:

  • {bought:0}, zero means cart that have not been purchased.
  • {bought:1}, one means cart that have been purchased.
  • There will be only one cart {bought:0} for client.
  • There is no restriction for the amount of cart {bought:1} for client.
  • User id and date{creation_at: 05-07-2024h15:37:29} must be provided.
  • A complete JSON contaning all fields and values must be passed to the API.

GET /cart

Returns user's cart.

Example Request:

GET https://next-store-api-62m2.onrender.com/cart/{USER ID}

Example Response 200:

{
        "items": [
        {
        "user_id": "4jgm2z1nr3lylryo11",
        "item_id": "8",
        "qnt": 2,
        "bought": 0,
        "price": 129.99,
        "name": "Chanel Coco Noir Eau De",
        "creation_at": "05-07-2024h15:37:29"
        },
          // ... more items ...
        ]
        }

Example Response 404: 

{message: "Not found"}



GET /cart purchased

Returns carts that have been purchased by the user.

Query Parameters:

  • user_id: User id
  • cart_id: Time that the cart was created {creation_at:05-07-2024h15:37:29}

Example Request:

GET https://next-store-api-62m2.onrender.com/cart/purchased/params?user_id=4jgm2z1bxslyle2o5n&cart_id=05-07-2024h15:37:29

Example Response 200:

{
        "items": [
        {
        "user_id": "4jgm2z1nr3lylryo11",
        "item_id": "8",
        "qnt": 2,
        "bought": 1,
        "price": 129.99,
        "name": "Chanel Coco Noir Eau De",
        "creation_at": "05-07-2024h15:37:29"
        },
        // ... more items ...
                 ]
  }

Example Response 404: 
{ message: "Not found" }



POST /cart

Create a new cart.

Example Request:

POST https://next-store-api-62m2.onrender.com/cart

Request Body:

{
        "id":"4jgm2z11gfjlyrfbmsl", //=> THIS IS THE USER ID 
        "item":{
        "id": "8",
        "quantity": "2",
        "createAt": "05-07-2024h15:37:29", //This is the ID of the cart in the database
        "price":"129.99",
        "name":"Chanel Coco Noir Eau De"
        }
}


Example Response 200: 

{ message: "Cart created successufuly" }

Example Response 407: 

{ message: "Incomplete Body" }
{ message: "There is no pruduct with id: xxxx or User with id: xxxx"}
{ message: "UNIQUE constraint failed: products.title"}



DELETE /cart

Delete cart.

Example Request:

DELETE https://next-store-api-62m2.onrender.com/cart

Request Body:

{
"cart":"4jgm2z1bxslyle2o5n", //=> THIS IS THE USER ID. 
"op":0                      //=> THE OPTION 0 DELETE THE CART.
}


Example Response 200: 

{ message: "Cart deleted" }

Example Response 407: 

{ message: "Incomplete Body" }
{ message: "Not found"}



DELETE /product from cart

Delete product from cart.

Example Request:

DELETE https://next-store-api-62m2.onrender.com/cart

Request Body:

{
  "cart":{
  "user_id":"4jgm2z1bxslyle2o5n",
  "item_id":"9"
  },
  "op":1 //=> THE OPTION 1 DELETE A PRODUCT FROM THE CART.
}


Example Response 201: 

{ message: "Cart deleted" }

Example Response 407: 

{ message: "Incomplete Body" }
{ message: "Not found"}
