Product Routes

Products route features and usage:

  • User submitted Product are erased on a regular basis.
  • Submitted Product must contain at least two images references.
  • Each product must contain an unique ID and TITLE.
  • A complete JSON contaning all fields and values must be passed to the API.

GET /filter

Returns products filtered.

Query Parameters:

POST/product

Create a new product.

Example request:

POST https://next-store-api-62m2.onrender.com/products

Request body:

{
"id":"234",
"title":"Super UNIQUE iPhone 334 ",
"description":"An apple mobile of the future",
"price":"549",
"discountPercentage":"12.96",
"rating":"4.69",
"stock":"94",
"brand":"Apple",
"category":"smartphones",
"thumbnail":"https://cdn.dummyjson.com/product-images/1/thumbnail.jpg",
"images":[
    "https://cdn.dummyjson.com/product-images/1/1.jpg",
    "https://cdn.dummyjson.com/product-images/1/2.jpg",
    "https://cdn.dummyjson.com/product-images/1/3.jpg",
    "https://cdn.dummyjson.com/product-images/1/4.jpg",
    "https://cdn.dummyjson.com/product-images/1/thumbnail.jpg"
    ]
}


Example response 201: 

{ message: "Product created" }

Example response 404 / 500: 

{ message: "UNIQUE constraint failed: products.id"}
{ message: "UNIQUE constraint failed: products.title"}
{ message: "Incomplete Body" }

Example request:
