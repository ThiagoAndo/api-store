User Routes

Users route features and usage:

  • Name must be passed as first name and last name only.
  • Password must contain at least eight character.
  • Email must be a valid email address format and unique.
  • Token is valid for one hour and is returned by the API with user id in a JSON object.
  • A complete JSON contaning all fields and values must be passed to the API.

POST /register

Registers a new user.

Example request:

POST https://next-store-api-62m2.onrender.com/user/new

Request body:

{
     "first_name": "Stephen",
     "last_name":"King",
     "email_address":"stephenking@lorem.com",
     "password":"12345678"
}

PATCH /user detail

Update user data.

Example request:

PATCH https://next-store-api-62m2.onrender.com/user

Request body:

{
    "id": "4o40o1vlyltmkxk",
    "first_name":"Stephen" , 
    "last_name":"King",
    "email_address":"stephenking@lorem.com"
}

Example response 201:

 { message: "Updated user detail with id XXXX"}

Example response 404 / 500: 
{ message: "Incomplete Body" }
{ message: "Could not update user with id XXXX"}
{ message: "This email has been used by an other user or user id is incorrect"}
{ message: "Name is not valid. Make sure to enter first and last name only"}
{ message: "Email is not valid"}



PATCH /user password

Update user password.

Example request:

PATCH https://next-store-api-62m2.onrender.com/user/password

Request body:

{
        "id": "4o40o1vlyltmkxk",
        "password":"87654321"
    }

Example response 201:

 { message: "Updated user detail with id XXXX"}

Example response 404 / 500: 
{ message: "Incomplete Body" }
{ message: "Could not update user with id XXXX"}
{ message: "Password must contain at least eight character"}



DELETE /user

Delete an existing user.

Example request:

DELETE https://next-store-api-62m2.onrender.com/user

Request body:

{"id": "4o40o1vlyltmkxk"}

Example response 200:

{ message: "Deleted user with id XXXX" }

Example response 404 / 500: 
{ message: "Incomplete Body" }
{ message: "Could not delete user with id XXXX"}
