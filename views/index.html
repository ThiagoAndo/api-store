Next Store API Documentation

Welcome to the Next Store API. The products endpoint provides a comprehensive dataset of sample product information, including details like names, prices, descriptions, images, and categories, ideal for testing and prototyping e-commerce applications. This API allows you to manage and retrieve products, users, users' address, users' carts and users' purchases. Please refer to the documentation below for details on how to interact with the API. This is a Restful API build using Node.js and Next.js. It also uses a package called better-sqlite3 that allows you to create a local relational database and query it using Node.js. Thus, some routes require the client to send one JSON object per request, which will be inserted into the table.

Note : You may experience a delay in server response due to the free hosting service provided by RENDER.

Key Features

The API provides endpoints for managing and retrieving:

  • Products
  • Users
  • Users' Addresses
  • Users' Carts
  • Users' Purchases

Authentication

  • Bearer Token Authentication is required for some endpoints. Use your email and password to get a token obtained from the /user/new endpoint.

POST /register

Registers a new user.

Example request:

POST https://next-store-api-62m2.onrender.com/user/new

Request body:

{
     "first_name": "Stephen",
     "last_name":"King",
     "email_address":"stephenking@lorem.com",
     "password":"12345678"
